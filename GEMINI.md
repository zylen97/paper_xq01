# 论文开发规则

本文件规定了论文撰写的规则和惯例，以确保一致性和高质量。

## 1. 写作风格

*   **目标期刊:** 所有内容都应以符合工程管理领域顶级SCI检索国际期刊的风格撰写。
*   **科技论文风格:** 在整个手稿中保持正式、客观和科学的基调。
*   **主动语态:** 写作时主要使用主动语态，使句子清晰直接。
*   **词汇:** 使用工程管理学科的标准和适当术语。
*   **句子结构:** 避免使用过于复杂的句式，优先使用清晰、直接的简单句和复合句。

## 2. 引用

*   当一个陈述或句子需要引用时，请在末尾添加 `(ref)` 进行标记。这将在后续被替换为正确的引文。

## 3. LaTeX 使用

*   遵循标准的 LaTeX 最佳实践来结构化文档。
*   确保所有图表都有标题并在文本中被引用。
